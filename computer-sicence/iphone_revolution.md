## 📊 技术演进关键趋势

  * **工艺制程** ：16nm (A10) → 10nm (A11) → 7nm (A12-A13) → 5nm (A14-A16) → 3nm (A17-A18) → 3nm N3P (A19)
  * **内存容量演进** ：2GB (A10) → 3-4GB (A11-A13) → 4-6GB (A14-A15) → 6-8GB (A16) → 6-12GB (A17) → 8-12GB (A18) → 8-12GB (A19 预测)
  * **AI 引擎核心数演进** ：无 → 2核 → 8核 → 16核（A17-A19）
  * **AI TOPS 性能演进** ：- → 0.6 TOPS → 1.0 TOPS → 11 TOPS (A14) → 16 TOPS (A15) → 17 TOPS (A16) → 22 TOPS (A17) → 35 TOPS (A18) → 50+ TOPS (A19)
  * **GPU 升级路线** ：PowerVR（3-4核） → Apple GPU（4-5核） → Apple GPU（5核 + 硬件光线追踪 @ A19）
  * **摄像头革新** ：12MP 主摄 → 48MP 主摄（A16/iPhone 14 Pro） → 48MP Fusion 双摄（A19/iPhone 17）→ 48MP Fusion + 超广角 + 18MP 中央舞台相机（iPhone 17 Pro）
  * **三主线驱动** ： 
    * **性能主线** ：芯片制程、CPU/GPU、AI 引擎持续升级
    * **存储主线** ：内存容量伴随 Pro 款式分化，普通款稳定在 4-8GB，Pro/Max 则升至 8-12GB
    * **影像主线** ：主摄分辨率、镜头光圈、传感器尺寸、算法优化协同演进
